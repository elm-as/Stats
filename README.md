## 🚀 Installation

Si vous souhaitez faire tourner le projet en local, voici comment procéder :

```bash
# 1. Cloner le projet
git clone https://github.com/elm-as/Stats.git
cd Stats

# 2. Installer les dépendances Backend
cd backend
python -m venv venv
# Activer l'environnement virtuel (Windows: venv\Scripts\activate, Linux/Mac: source venv/bin/activate)
pip install -r requirements.txt

# 3. Installer les dépendances Frontend
cd ../frontend
npm install

# 4. Lancer l'environnement de développement
# Backend (dans un terminal)
flask run

# Frontend (dans un autre terminal)
npm run dev
```
*(N'oubliez pas de configurer vos variables d'environnement pour Supabase et l'IA dans un fichier `.env`)*
